🧠 CONCEPTO: FULL JOIN
El FULL JOIN (o FULL OUTER JOIN) sirve para:

👉 mostrar TODOS los datos de ambas tablas
aunque no tengan relación entre sí.

🎯 QUÉ INCLUYE
✔ coincidencias (como INNER)
✔ los que están solo en la izquierda
✔ los que están solo en la derecha

🧠 VISUAL MENTAL
LEFT + RIGHT = FULL JOIN

/*************************************/
En este caso se usa FULL JOIN
SELECT ...
FROM A
LEFT JOIN B ON ...

UNION

SELECT ...
FROM A
RIGHT JOIN B ON ...
/*************************************/
