🧠 SEMANA 11 – Funciones agregadas + GROUP BY + HAVING

/**************************************/
🔹 1. ¿Qué son las funciones agregadas?
/**************************************/
Son funciones que resumen datos de varias filas en un solo resultado.

👉 Se usan mucho en reportes (promedios, totales, conteos, etc.)

📌 Principales:
SUM() → suma valores
COUNT() → cuenta registros
AVG() → promedio
MAX() → valor máximo
MIN() → valor mínimo

/********************************************************/
🔹 2. Ejemplo mental (tu base de datos de Guardian Tale)
/********************************************************/
👉 “Quiero saber el promedio de ataque de todos los personajes”

SELECT AVG(ataque) FROM personajes;

👉 “Cuántos personajes hay”

SELECT COUNT(*) FROM personajes;

👉 “El ataque más alto”

SELECT MAX(ataque) FROM personajes;

/*****************************/
🔹 3. GROUP BY (AGRUPAR DATOS)
/*****************************/
Sirve para agrupar filas con características iguales.

👉 Ejemplo clave:

“Promedio de ataque por rareza”

SELECT rareza, AVG(ataque)
FROM personajes
GROUP BY rareza;

💡 Aquí pasa esto:

Agrupa por NR, SR, SSR
Calcula el promedio dentro de cada grupo

/******************************/
🔹 4. HAVING (FILTRO DE GRUPOS)
/******************************/
👉 Es como el WHERE, pero aplicado a grupos

⚠️ IMPORTANTE:

WHERE → antes de agrupar
HAVING → después de agrupar

📌 Ejemplo:

“Mostrar solo rarezas con promedio de ataque mayor a 180”

SELECT rareza, AVG(ataque) AS promedio
FROM personajes
GROUP BY rareza
HAVING promedio > 180;

/*****************************************/
🔹 5. Diferencia clave (MUY IMPORTANTE 🔥)
/*****************************************/
WHERE	HAVING

Filtra filas	      Filtra grupos
Antes del GROUP BY	  Después del GROUP BY
No usa agregados	  Usa funciones como AVG, SUM

/***********************************/
🔹 6. Ejemplo COMPLETO (tipo examen)
/***********************************/

👉 “Cantidad de personajes por elemento, pero solo los que tengan más de 10 personajes”

SELECT id_elemento, COUNT(*) AS total
FROM personajes
GROUP BY id_elemento
HAVING total > 10;
